# AeroTwinFlight
유니티 3D 제작 두 번째 포트폴리오 프로젝트

## 프로젝트 소개
* **개발 기간**: 2026.06 ~ 진행 중
* **개발 환경**: Unity 3D
* **개발 인원**: 1인 (kacchi)

## 주요 기능 (구현 예정)
* 3D 비행기 조작 및 카메라 시스템
* 프레임 매니저 및 최적화 시스템
* UI 연동
